상속 (inheritance)

html{color: red;}
-> html의 하위 태그(body, ul, h1, h2, li, p 등..)에 상속됨
body{border:1px solid green;}
-> border속성은 상속되지 않음

수업내용

  • html
  • css
  • Javascript
Full Property Table 를 확인하면 inheritanced?에서 상속되는 속성인 지 아닌 지 알 수 있다.
ex) border: NO, color: YES
