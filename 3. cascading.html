캐스케이딩(cascading)이란??

      본래 뜻은 '위에서 아래로 흐르는', 상속 또는 종속하는' 이라는
      의미를 갖고 있다. 이는 선택자에 적용된 여러 스타일 중

수업내용

  • HTML
  • CSS
  • Javascript
      위의 코드는
      
      <style>
        li{color:red;}  (4순위)
        #idsel{color:blue;}  (2순위)
        .classsel{color:green;}  (3순위)
      </style>

      <li id="idsel" class="classsel" 
      style="color:powderblue">CSS</li> (1순위)
      로 되어있다

캐스케이딩 (스타일 우선순위)

구체적/명시적 일수록 우선순위가 높다

  1. !important (우선순위 강제 ↑)
  2. style attribute
  3. id selector(id는 중복x)
  4. class selector
  5. tag selector
